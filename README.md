## game format

Displayed example:

Where `-` is none, `*` is possible move, and `X` and `O` are p1 and p2 respectively

```
X--O----
X--OOOO-
X--OX*--
XXOXO*--
XXXOXO-*
XXOO*XXX
X-OOX*-O
--O-O*--
```

Compressed format:

```
O:X--O----X--OOOO-X--OX---XXOXO---XXXOXO--XXOO-XXXX-OOX--O--O-O---
```
